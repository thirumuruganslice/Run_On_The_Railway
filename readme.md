## Subway Surfers - Run on the Railway

A fast-paced 3D endless runner inspired by subway chases. Dash along railway lanes, dodge trains and barriers, collect coins, and chase high scores. Built with Three.js and a custom game loop.

## Controls

- Left/Right arrow: switch lanes
- Up arrow: jump
- Down arrow: roll
- P: pause
- On mobile: swipe to move

## Features

- 3D endless runner with lane switching, jumping, and rolling
- Dynamic obstacles, trains, coins, and power-ups
- Score, distance, and coin tracking with a HUD
- Web Audio API synth sound effects
- Stylized UI with start, pause, and game over screens

## Tech Stack

- HTML5 Canvas
- Three.js (r128 via CDN)
- Vanilla JavaScript
- CSS for HUD and menus

## Project Structure

- index.html - entry point and UI structure
- css/style.css - HUD and screen styles
- js/game.js - game engine, mechanics, and rendering
